DS_HEAP&STACK_PopSequence

4.5h

Description

Given a stack which can keep M numbers at most. Push N numbers in the order of 1, 2, 3, ..., N and pop randomly. You are supposed to tell if a given sequence of numbers is a possible pop sequence of the stack. For example, if M is 5 and N is 7, we can obtain 1, 2, 3, 4, 5, 6, 7 from the stack, but not 3, 2, 1, 7, 5, 6, 4.

Input Specification:

Each input file contains one test case. For each case, the first line contains 3 numbers (all no more than 1000): M (the maximum capacity of the stack), N (the length of push sequence), and K (the number of pop sequences to be checked). Then K lines follow, each contains a pop sequence of N numbers. All the numbers in a line are separated by a space.

Output Specification:

For each pop sequence, print in one line "YES" if it is indeed a possible pop sequence of the stack, or "NO" if not.

Sample Input:

5 7 5
1 2 3 4 5 6 7
3 2 1 7 5 6 4
7 6 5 4 3 2 1
5 6 4 3 7 2 1
1 7 6 5 4 3 2

Sample Output:

YES
NO
NO
YES
NO

  2. 数组赋初值

    超链接为参考博客

    数组必须指定长度，而其数组定义的长度不能是变量或者浮点数，一定是常量表达式

    1. 全局数组

      全局情况下int默认初始值为0，double默认初始值为0

    2. 非全局数组

      非全局情况int默认都是垃圾值

      1. C/C++不支持数组的整体赋值，只能声明数组时整体初始化，并且只能初始化为0

      2. 若整体初始化为其他数字比如1，则只有第一位是1，其余均为0

      3. 若要初始化为其他数组，则用for循环手动赋值，或者memset和fill函数

        1. memset（一般只能用来填充char型数组）
          1. 按照字节填充某字符
          2. 在头文件<cstring>里面
          3. 使用方法memset(数组名，要填充的内容，单元大小)
        2. fill（可以赋值任何）（C++）
          1. 按照单元赋值，将一个区间的元素都赋同一个值
          2. 在头文件<algorithm>里面
          3. 使用方法：fill(起始地址,结束地址, 要填入的内容)；
        //非全局情况
char p[10]={'1'};
int pp[10]={1};
char d[10];
memset(d,'1',sizeof(d));
//结果
1

    3. 字符数组的特殊处理办法

      超链接为参考博客

      1. 定义时用直接字符串赋值

        char a[10]="hello";

char a[10]; a[10]="hello";//错误！一个字符怎么能容纳一个字符串？况且a[10]也是不存在的！

      2. 对数组中字符逐个赋值

        char a[10]={'h','e','l','l','o'};

      3. 利用strcpy

        char a[10]; strcpy(a, "hello");

      4. 常见错误

        char a[10]; a[10]="hello";//一个字符怎么能容纳一个字符串？况且a[10]也是不存在的！
char a[10]; a="hello";//这种情况容易出现，a虽然是指针，但是它已经指向在堆栈中分配的10个字符空间，现在这个情况a又指向数据区中的hello常量，这里的指针a出现混乱，不允许！

// C语言的运算符根本无法操作字符串。在C语言中把字符串当作数组来处理，因此，对字符串的限制方式和对数组的一样，特别是，它们都不能用C语言的运算符进行复制和比较操作。直接尝试对字符串进行复制或比较操作会失败。
char str1[10], str2[10];
//利用=运算符来把字符串复制到字符数组中是不可能的：
str1 = "abc";     /*** WRONG ***/
str2 = str1;       /*** WRONG ***/
//C语言把这些语句解释为一个指针与另一个指针之间的（非法的）赋值运算。但是，使用=初始化字符数组是合法的：
 char str1[10] = "abc";
//这是因为在声明中，=不是赋值运算符。
//试图使用关系运算符或判等运算符来比较字符串是合法的，但不会产生预期的结果：
if (str1==str2) ...    /*** WRONG ***/
//这条语句把str1和str2作为指针来进行比较，而不是比较两个数组的内容。因为str1和str2有不同的地址，所以表达式str1 == str2的值一定为0。

Notice

  1. 堆栈下标一类的问题
  2. 在读取一行输入时，如果中途发现有错误，可以用fgets吃掉一整行，然后扔掉这部分空间
  3. 循环变量不要滥用i，j！用之前要注意看一下会不会导致错误
  4. 一些全局变量若被不同情况下重复使用（比如pop sequence中的lastpush 以及Stack[n]），开启新的循环时要先将全局变量初始化

//Pop Sequence
//M (the maximum capacity of the stack)
//N (the length of push sequence)
//K (the number of pop sequences to be checked)
#include <stdio.h>
#include<stdlib.h>
#define MAXSIZE 1000
int Stack[MAXSIZE+1];
int M,N,K;
int instack = 0;
int popped[MAXSIZE+1]= {0};
int ans[MAXSIZE+1];
int lastpush = 0;
int top = 0;
char buffer[2*MAXSIZE];
int isfull(int n);//0-->unfilled;1-->full
void push(int n);
void pop();
void Print();
int main()
{
    int i,j,t,n,flag;//ans=0-->NO;and=1-->YES
    scanf("%d %d %d",&M,&N,&K);
    for(i = 0; i < MAXSIZE+1; i++)  ans[i] = 1;
    for(i = 0; i < K; i++)
    {
        lastpush = 0;
        top = 0;
        instack = 0;
        for(t = 0; t < MAXSIZE+1; t++)
        {
            popped[t] = 0;
            Stack[t] = 0;
        }
        for(j = 0; j < N; j++)
        {
            scanf("%d", &n);
            if(top == 0)
            {
                instack = 0;
                flag = isfull(n);
                if(!flag)
                {
                    //top++;
                    push(n);
                    pop(top);
                }
                else
                {
                    ans[i] = 0;
                    fgets(buffer,MAXSIZE,stdin);
                    break;
                }
            }
            else
            {
                if(n<Stack[top-1])//不可能同一批先入栈的先出
                {
                    fgets(buffer,MAXSIZE,stdin);
                    ans[i] = 0;
                    break;
                }
                else if(n == Stack[top-1])
                {
                    pop(top);
                }
                else if(n > Stack[top-1])
                {
                    flag = isfull(n);
                    if(!flag)
                    {
                        push(n);
                        pop();
                    }
                    else
                    {
                        fgets(buffer,2*MAXSIZE,stdin);
                        ans[i] = 0;
                        break;
                    }
                }
                
            }
        }
    }
    Print();
    system ("pause");
    return 0;
}
int isfull(int n)
{
    int flag = 0;
    if(instack == 0)
    {
        //int i = n;
        //while(popped[i] == 0) i--;
        if(n - lastpush > M)    flag = 1;
    }
    else    
    {
        int tmp = Stack[top-1];
        int gap = 0;
        while(tmp<n)//去掉已经被弹出的元素
        {
            if(popped[++tmp])   gap++;
        }
        if(instack + n - Stack[top-1] - gap > M)    flag = 1;
    }
    return flag;
}
void push(int n)
{
    int i;
    for(i = lastpush+1; i <= n; i++)
    {
        Stack[top++] = i;
        instack++;
    }
    lastpush = n;
    return ;
}
void pop()
{
    int tmp = Stack[--top];
    Stack[top] = 0;
    popped[tmp] = 1;
    instack--;
    return ;
}
void Print()
{
    int i;
    for(i = 0; i < K-1; i++)
    {
        if(ans[i]==1)   printf("YES\n");
        else    printf("NO\n");
    }
    if(ans[i]==1)   printf("YES");
    else    printf("NO");
    return ;
}

Version-2

参考了这篇博客的代码，改进了以下几点

  1. 不需要instack， 直接看top就可以了
  2. 不用top的值作为分类标准，用出栈元素与栈顶值的大小关系分类
  3. 相比该博主的代码 ，我改进了读入的操作，是边读边判断，而不是一行全部先读入再逐个判断，这样更省时间(可能？)。但是要注意在一行内中断读入后，要用fgets舍弃这一行，才能从下一行开始读入
  4. 原来的buffer开成全局变量有问题，改为第一层循环内的局部变量了
  5. 这里两个判断语句的前后顺序还是很容易出错的，比如“5 6 4 3 7 2 1”这组输入，如果用如图的顺序判断就会被当作堆栈溢出报错，而顺序换一下的话，原来栈顶元素5被弹出后再压入6，栈空间就不会溢出

  1. 另外，因为*处忘记了这段语句还卡了半天没跳出来，以后注意，要么一行行读进来再判断，要么记得更新读入的位置
fgets(buffer, MAXSIZE,stdin);
#include<stdio.h>
#include<stdlib.h>
#define MAXSIZE 1000

int main()
{
    int M,N,K;
    int i,j,k;
    scanf("%d %d %d",&M,&N,&K);
    for(i = 0; i < K; i++)
    {
        int top = -1;
        int Stack[MAXSIZE] = {0};
        int num = 0;
        char buffer[2*MAXSIZE];
        Stack[++top] = ++num;
        for(j = 0; j < N; j++)
        {
            int input;
            scanf("%d", &input);
            if(input<Stack[top])
            {
                fgets(buffer, MAXSIZE,stdin);
                break;
            }
            else if(input > Stack[top]) while(num<input)    Stack[++top] = ++num;
            if(top>=M){
                fgets(buffer, MAXSIZE,stdin);
                break;
            }
            if(input == Stack[top]) top--;
            if(top<0)   Stack[++top] = ++num;
        }
        if(i<K)
        {
            if(j==N)    printf("YES\n");
            else    printf("NO\n");
        }
        else
        {
            if(j==N)    printf("YES");
            else    printf("NO");
        }
    }
    return 0;
}
