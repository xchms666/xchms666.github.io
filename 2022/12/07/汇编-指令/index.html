PUSHDF,POPF

背景：push pop 不能直接操作FL寄存器（实际上任何指令都不能）-

  • 应用举例
  • 标志寄存器FL的相关指令
  • 要让TF=1（TF是FLAG的第八位)
    • 不存在相关指令
      • 因为它使用频率很低
  • 举例——让Flag的第八位清零
  • 32位系统中
    • 寄存器EFL
    • pushfd&popfd
    • 此时pushf&popf只控制EFL的低16位即FL

符号扩充指令：CBW, CWD, CDQ

只针对符号数

  • 用途

  • 对象
    • cbw al-> ax
    • cwd ax->dx:ax
    • cdq eax->edx:eax

负数左加F，正数右加0

  • 应用举例

主要为除法服务

零扩充指令：movzx

mov by zero extension

  • 目标操作数和原操作数可以不相关
  • 可以把八位直接扩充成32位

符号扩充指令：movsx

  • movsx ax，al ；相当于cbw
  • 自此cbw等不太用了

imul指令

第一类用法：和mul一样

第二类用法：可以包含2~3个操作数

换码指令：XLAT

translate 也称查表指令

  • 应用举例

实现十六进制的转化


.386 ; 表示程序中会用32位的寄存器
data segment use16; use16表示偏移使用16位
t db "0123456789ABCDEF"
x dd 2147483647
data ends

code segment use16
assume cs:code, ds:data
main:
   mov ax, data    ;\
   mov ds, ax      ; / ds:bx->t[0]
   mov bx, offset t;/
   mov ecx, 8
   mov eax, x
next:
   rol eax, 4
   push eax
   and eax, 0Fh
   xlat
   mov ah, 2
   mov dl, al
   int 21h
   pop eax
   sub ecx, 1
   jnz next
   mov ah, 4Ch
   int 21h
code ends
end main

XLAT代替了这么多行代码
