Created2022-11-09|Updated2022-11-09|汇编
|Post View:

设置指令断点

F2 breakpoint

bl breakpoint list

  • 显示目前已有断点信息

  • C = 01断点发生一次就停下来

bc breakpoint cleaner

  • bc 1 清除一号断点
  • bc * 清除全部断点

x execute

  • 快速运行，停在断点

设置硬件断点

在变量地址处设置断点

bpmb cs:3 W

  • breakpoint on memory byte at cs:3 when it is wirtten
  • R-被读取时
  • RW-被读取或写入时
  • X-执行到时
    • 此时也会亮起来
  • 停在写入指令的下一条指令

可以用bl来显示两种断点

  • BPMB 硬件断点
  • BPX 指令断点

bpmw cs:3 W

  • breakpoint on memory word

bpmd cs:3 W

  • breakpoint on memory dword

bpx断点和bpmb断点区别

  • bpx数量可以很多，bpmb最多四个

    • 因为CPU里面只有四个硬件断点寄存器

  • bpx will change 1st byte to 0xCC(CC int 3,i.e. software breakpoint)
