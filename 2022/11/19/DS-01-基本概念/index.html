DS_01_基本概念

算法

定义

  • 一个有限指令集
  • 接受一些输入
  • 产生输出
  • 一定在有限步骤后终止
  • 每条指令必须
    • 有充分明确的目标，不可以有歧义
    • 计算机能处理的范围之内
    • 描述不依赖于任何计算机语言或具体实现手段

评判

S（N）

  • 递归调用时，空间随N线性增长

void mergesort(int i,int j){
    int m;
    if(i != j){
        m = (i+j)/2;
        mergesort(i,m);
        mergesort(m+1,j);
        merge(i,j,m); //本函数的时间复杂度为O(n)
    }
}

最大子列和

算法一 O（N3）

#include<stdio.h>
int MaxSubSeq1(int n, int *arr)
{
    int i,j,k;
    int MaxSum,ThisSum = 0;
    MaxSum = arr[0];
    for(i = 0; i < n; i++)
    {
        for(j = 0; j < n; j++)
        {
            for(k = i; k <= j; k++)
            {
                ThisSum += arr[k];
                if(ThisSum > MaxSum)    MaxSum = ThisSum;
            }
            ThisSum = 0;
        }
    }
    return MaxSum;
}

int main()
{
    int sum;
    int a[8] = {0,12,-2,23,-4,-9,20,-7};
    sum = MaxSubSeq1(8, a);
    printf("%d\n",sum);
    return 0;
}

  • k从i开始，而不是从0开始
